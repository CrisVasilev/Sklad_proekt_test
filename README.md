# Система за складова наличност

## Описание

Това е Python проект за управление на складова наличност.

Програмата позволява:
- добавяне на продукти;
- показване на налични продукти;
- търсене на продукт;
- записване във файл;
- автоматично зареждане на данни.

## Използвани технологии

- Python
- OOP (Object Oriented Programming)
- Работа с файлове
- Colorama библиотека

## Използвани библиотеки

- colorama

## Структура на проекта


main.py
models.py
products.txt
requirements.txt
README.md


## Стартиране

Инсталиране на библиотеките:

pip install -r requirements.txt

Стартиране на програмата:

python main.py
